Netwerk

Schakel SSL-verificatie uit als je achter een corporate proxy zit die HTTPS inspecteert met een eigen root-CA. De GitHub sync werkt dan ook als de bedrijfs-CA niet in de Windows store staat.

GitHub-authenticatie

Personal Access Token met repo-scope. Nodig om vanuit het dashboard bestanden naar de repo te pushen. Lokaal opgeslagen in settings.json (die staat in .gitignore — wordt dus niet mee-gepusht). Laat leeg om te vertrouwen op de omgevingsvariabele GITHUB_TOKEN.

Runs-opslag

Output van geplande runs wordt per run als .json én leesbare .txt naar deze map geschreven (bestandsnaam: YYYY-MM-DD_HH-MM-SS_<naam>_<status>). Laat het pad leeg om de standaardlocatie naast het dashboard te gebruiken. Geef een absoluut pad op (bijv. naar een OneDrive-map of netwerkshare) als een andere automation de bestanden later moet oppikken.
